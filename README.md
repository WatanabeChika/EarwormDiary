# 耳虫日记 (Earworm Diary)

> **捕捉你每日清晨脑海中的那段旋律。**  

## 简介

**耳虫日记** 是一款专注于记录“耳虫现象”（Earworm）的 Android 应用程序。

你是否也有过这样的经历：清晨醒来，脑海中莫名循环着一段旋律？或者走在路上，突然被某首歌“洗脑”？这款 App 旨在帮助你捕捉这些稍纵即逝的听觉记忆。它不仅仅是一个记录工具，更是一本由旋律编织而成的私人日记。

## 核心特性

### 可视化旋律日历

告别枯燥的文字列表。我们以**专辑封面**为核心，将你的月度音乐心情拼凑成一幅画卷。

* **月视图总览**：直观呈现每天的“耳虫”封面。
* **快速回溯**：点击任意日期，即可查看当天的详细记录与心情。
* **年份/月份跳转**：支持快速时间切换，轻松穿越回过去的某个音乐瞬间。

### 高效交互与记录

为记录而生的流畅体验，让捕捉灵感不再繁琐。

* **双模记录**：支持扫描检索本地曲库，也支持直接输入纯文本记录（当脑海中的旋律不知其名时）。
* **拖拽复制**：独创的**长按拖拽**手势，只需在日历格子上轻轻一拖，即可将昨天的旋律“延续”到今天。
* **空状态引导**：精心设计的引导页，让每一天的开始都充满仪式感。

### 离线优先与隐私

你的记忆只属于你自己。

* **本地索引**：极速扫描设备存储，自动解析音频文件的元数据（封面、歌手、歌名）。
* **离线存储**：所有日记数据完全保存在本地（JSON/SharedPrefs），无需联网，无云端上传，绝对隐私安全。

### 现代 Material Design

* **纯净界面**：无广告、无干扰的沉浸式设计。
* **丝滑动画**：基于 Jetpack Compose 构建，提供流畅的转场与交互反馈。
* **精致 UI**：圆角卡片、柔和阴影与动态布局，带来愉悦的视觉享受。
