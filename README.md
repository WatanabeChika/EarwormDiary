# 🎵 耳虫日记 (Earworm Diary)

> **捕捉每日清晨，脑海中盘旋的那段旋律。**

## 📖 简介

**耳虫日记** 是一款专注于记录“耳虫现象”（Earworm）的 Android 应用程序。

你是否也有过这样的时刻：清晨醒来，脑海中莫名循环着一段旋律？或是行色匆匆时，突然被某首歌“击中”心绪？这款 App 旨在帮你捕捉这些稍纵即逝的听觉记忆。它不仅是一个记录工具，更是一本由旋律与专辑封面编织而成的**听觉手账**。

## ✨ 核心特性

### 📅 沉浸式音乐日历

告别枯燥的列表，用**专辑封面墙**拼凑你的月度音乐心情。

* **月视图画卷**：以封面图填充日历，直观呈现每日“耳虫”。
* **时光回溯**：点击日期查看详细记录，支持年份/月份快速跳转，穿越回过去的某个音乐瞬间。
* **多维分类**：支持自定义标签与分类管理，让你的音乐偏好有迹可循。

### ⚡ 高效交互与智能记录

为捕捉灵感而生的流畅体验，让记录不再繁琐。

* **混合检索模式**：
  * **本地扫描**：极速索引设备内的高音质音频。
  * **联网搜索**：集成网易云音乐 API，本地无资源时也可精准锁定网络歌曲。
  * **纯文本兜底**：不知其名时，支持快速记录脑海中的旋律片段。


* **智能关联修复**：在导入数据或刷新曲库时，自动将过往的“纯文本”或“网络记录”匹配至本地新下载的高音质文件，实现数据“无感升级”。
* **手势复刻**：独创**长按拖拽**交互，只需轻轻一拖，即可将昨天的旋律“延续”到今天。
* **仪式感引导**：精心设计的空状态页，让每一天的记录都充满期待。

### 🛡️ 离线优先与绝对隐私

你的记忆只属于你自己。

* **本地化索引**：无需上传文件，自动解析本地音频元数据（封面、歌手、歌名）。
* **数据主权**：所有日记数据完全存储于本地（JSON/SharedPrefs），无云端上传，无追踪代码，安全无忧。
* **便捷迁移**：支持标准 JSON 格式的导入与导出，备份随心。

### 🎨 现代 Material Design

* **纯净体验**：无广告、无干扰的沉浸式设计。
* **丝滑动画**：基于 **Jetpack Compose** 构建，带来流畅的转场与跟手反馈。
* **精致 UI**：动态取色、圆角卡片与柔和阴影，带来愉悦的视觉享受。
